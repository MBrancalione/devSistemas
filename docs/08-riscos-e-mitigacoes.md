# 8. Riscos e Mitigações

| Risco Identificado | Estratégia de Mitigação / Ação |
| :--- | :--- |
| **Reações alérgicas a cosméticos** | Aquisição junto a fornecedores que seguem critérios de cosmetovigilância. Orientação para o cliente contatar o fabricante e a ANVISA caso ocorra. |
| **Produtos sem registro (ANVISA)** | Verificação minuciosa do status regulatório de cada SKU antes da comercialização. |
| **Ocorrências logísticas (extravios/atrasos)** | Sistemas de rastreamento avançados, seleção criteriosa de transportadoras e contratação de seguros de carga. |
| **Imprevisibilidade tributária** | Monitoramento constante de portarias e taxas aduaneiras de importação. |
| **Violação de dados (Segurança)** | Investimento em firewalls, criptografia, protocolos de segurança robustos e total conformidade com a LGPD. |
| **Falha na integração de pagamentos** | Exibição de mensagens claras de erro, opção de troca de método de pagamento e retenção de logs para auditoria. |
| **Inconsistência no estoque concorrente** | Implementação de bloqueio de concorrência (travamento de registro) no banco de dados e testes de carga. |
| **Incompatibilidade de Navegadores** | Realização de testes cruzados (*cross-browser testing*) conforme estipulado no requisito RNF01. |
