# 2. Descrição do Projeto

O Eva N' Essence é um sistema web projetado para modernizar e digitalizar as operações de uma perfumaria, ampliando seu alcance comercial e melhorando a eficiência operacional.

## 2.1 Visão Geral do Projeto
A plataforma oferece aos clientes uma experiência de compra online completa, com catálogo categorizado, carrinho de compras, pagamento integrado e acompanhamento de pedidos. 

Para os funcionários, disponibiliza um portal administrativo dedicado para gestão de produtos, preços e estoque. 

Entre os principais benefícios estão: 

* A ampliação do alcance comercial da perfumaria;
* A automação do controle de estoque;
* A melhora na experiência do consumidor;
* O aumento da eficiência operacional.

### 2.2 Stakeholder
| Stakeholder | Papel |
| :--- | :--- |
| Equipe de Desenvolvimento | Responsáveis pelo levantamento de requisitos, modelagem, prototipação e desenvolvimento do sistema. |
| Professor Orientador | Responsável por orientar a equipe durante o desenvolvimento, estabelecer os critérios de avaliação e validar a entrega final. |
| Funcionários da Perfumaria | Devem fornecer feedback sobre a usabilidade das funcionalidades de gestão. |
| Clientes Finais | Suas necessidades de usabilidade e experiência de compra serão utilizadas para as decisões de design do sistema. |
