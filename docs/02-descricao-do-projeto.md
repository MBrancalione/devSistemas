# 2. Descrição do Projeto

O Eva N' Essence é um sistema web projetado para modernizar e digitalizar as operações de uma perfumaria, ampliando seu alcance comercial e melhorando a eficiência operacional.

## 2.1 Visão Geral do Projeto
A plataforma oferece uma experiência de compra completa para os clientes e disponibiliza um portal administrativo dedicado para os funcionários gerenciarem produtos, preços e estoque.
