# Bem-vindo à documentação do Eva N' Essence 
O projeto **Eva N' Essence** consiste em um sistema web de gestão e e-commerce voltado para perfumaria, integrando: 

- Controle de estoque 
- Catálogo online 
- Carrinho de compras 
- Pagamentos 
- Histórico de pedidos 
- Portal administrativo
